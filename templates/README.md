# 模板图片目录

此目录用于存放模板匹配所需的模板图片。

## 命名规范

- 使用小写英文字母和下划线
- 使用描述性名称，如 `run_button.png`、`save_icon.png`
- 避免使用空格和特殊字符

## 如何准备模板图片

1. **截取目标元素**
2. 
   - 使用截图工具截取目标 UI 元素
   - 尽量只包含元素本身，周围留少量边距（2-5 像素）

2. **保存到本目录**
   - 将截图保存为 PNG 格式
   - 使用描述性文件名

3. **在操作配置中引用**
   ```yaml
   - name: click_run_button
     template: run_button.png
     intent: template_match
   ```

## 最佳实践

- **保持简洁**：模板应只包含目标元素的核心特征
- **避免变化元素**：不要包含动态文本、状态指示器等
- **测试匹配**：使用不同截图测试匹配准确性
- **多尺度适配**：如果 UI 缩放可能导致问题，可启用多尺度匹配

## 目录结构

```

```
